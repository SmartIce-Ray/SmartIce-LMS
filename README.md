# SmartIce LMS

有点东西餐饮集团培训管理系统

## 项目结构

| 目录 | 说明 | 负责人 |
|-----|------|-------|
| lms-frontend/ | 前端 (Vue 3) | SmartIce-Ray |
| lms-backend/ | 后端 (FastAPI) | JeremyDong22 |

## 技术栈

- **前端**: Vue 3 + Vite + shadcn-vue + Tailwind CSS
- **后端**: Python + FastAPI + SQLAlchemy
- **数据库**: PostgreSQL

## 开发指南

### 前端

```bash
cd lms-frontend
pnpm install
pnpm dev
```
